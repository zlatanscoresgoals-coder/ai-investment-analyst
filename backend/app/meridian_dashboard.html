Enter a US ticker to pull the latest 10-K filing links and up to three years of annual financial metrics from SEC EDGAR (Company Facts / XBRL where available). This does not run the investor-lens score. To score any ticker and show it on Meridian, use POST /analysis/run-any/{ticker} (or add rows via API), then Run Full Analysis on the Meridian tab.

Live quotes refresh on each load. Scores and lens output come from the latest SEC annual metrics in the database and are updated when you run analysis (new 10-K years change the picture over time). The starter list is a curated default—add any US ticker with POST /analysis/run-any/{ticker}, then Run full analysis rescans every company in the database so names can drop to watchlist or rise to recommended as fundamentals and the blended framework evolve.
